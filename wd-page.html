Actionable Search Results

Compressing a multi-page action into a smooth interaction that lets users get out ow Workday and back towork quicker.

Demo

What/Why?

Workday People Experience is a version of Workday that is optimized for the everyday employee.

The everyday employee has a job to get back to. They want to get out of workday as soon as they can.

Actionable search result allow users to complete an activity from the search result page instead of clicking through to a report or application.

For the 2019 Workday Global Hackathon I teamed up with engineers across Workday to design and build an actionable search results card. Workday Assistant had conversational features that allowed users to make time off requests from within the chat interface.

Assistant's NLP capabilities allowed users to type in a phrase like "time off", and assistant would offer to execute a time off request. Our team added Assistant's NLP functionality as a search pre-processor that allowed us to surface Workday Assistant suggested results, and a interface for creating a time off request without needing to click through and load a new time-off report.
